lightweight http client for web and node

## Highlights

- Written in Typescript

## Installation

npm:

```shell
$ npm install xeta
```

yarn:

```shell
$ yarn add xeta
```

## Usage

> lightweight http client for web and node

```js

  const { xeta } = require('xeta');

  xeta(); // hello universe

```
